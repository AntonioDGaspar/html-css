Algumas Outras Formatações

As Tags Code e Pre

A tag Code serve para deixar um texto monoespaçado e mais legível, usada normalmente para códigos. Exemplo: let num = 1;

Para linhas de código com identação e/ou textos com quebras de linhas podemos usar a tag pre. 

  
let num = 1;
let num1 = 2;
let soma = num1 + num2;

As Tags Q e Blackquote

A tag Q serve para fazer citações simples e curtas. Exemplo: A vida é uma life.

A tag Blockquote permite fazer citações mais complexas, com direito à avanço no texto. Exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Aperiam placeat autem, laboriosam dolores dolore est neque tempore consequatur eaque ratione, similique nobis consequuntur nostrum voluptas. Laudantium doloribus consequatur nam fuga!

As Tags Abbr e Bdo

A tag Abbr serve para mostrar abreviações. Exemplo: HTML

A tag Bdo inverte textos (Pode ser da esquerda-direita e direita-esquerda). Exemplo: Rasen Rengan
