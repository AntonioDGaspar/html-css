Principais Formatações

Negrito / Destaque

O Negrito ou Destaque pode ser usado com a tag B, como está sendo neste texto, porém ela não tem significado, apenas afeta a forma do texto.

O jeito mais certo é utilizando a tag STRONG, para realçar um Termo Forte , este destaque não é apenas estético, ele também indica ao código que um termo importante está a ser destacado.

Itálico ou Enfase

Itálico pode ser usado com a tag I, assim como está aqui (Sem Semântica).

É possível usar também a tag EN para dar enfâse à um termo (Com Semântica).

Texto Marcado

Usando a tag MARK podemos marcar o texto.(Sem Semântica)

Texto Grande e Pequeno

Criando um Texto Grande com a tag morta BIG.

Criando um Texto Pequeno com a tag que SMALL, que ainda não morreu.

Textos Excluídos

Podemos criar textos Excluídos usando a tag DEL (Semântica)

Texto Inserido

Podemos criar textos Sublinhaods com a tag INS. (Semântica)

Podemos usar também a tag U . (Não semântica)

Textos Sobescrito

É possível usar a tag SUP para fazer algo como: x 3+0.

Textos Subescritos

Também é possível usar a tag SUB para algo como: H2O.
